# 思路分析

我看了一下题目感觉更像是找规律(不知道是不是错觉),我先穷举，发现2-2,3-3，4-5,5-8 2、3、5、8...不得了 这不是传说中的斐波那数列嘛 然后就按这个写一下，结果真的通过了，

思考：为什么是斐波那数列呢，思考一下就知道，比如要满足每次+1、+2 那么最后一步(n)肯定是1或者2,那不正是n-1阶楼梯(所需要的步数)走一步爬完和n-2阶楼梯(所需要的步数)走两步爬完，所以fn(n-2)+fn(n-1)满足斐波那数列的定义了。

# AC 代码

/**
 * @param {number} n
 * @return {number}
 */
var climbStairs = function(n) {
    let fb = (n)=>{
        let num1 = 1;
        let num2 = 1;
        let sum = num2;
        for(let i=1;i<n;i++){
            sum = num1+num2
            num1 = num2
            num2 = sum
        }
        return sum
    }
    return fb(n)
};
